Boopress

A lightweight Wordpress theme based on Bootstrap from Twitter.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Boopress Theme.

Boopress is a Wordpress theme for users and developers that it's simple and clean. Boopress built using HTML5 and Bootstrap structure for Wordpress 3.1 or great.

Share your posts in social network

Boopress have buttons for main social networks like Twitter, Facebook or Google Plus. In the theme options you can disable it.

Top and bottom menus

You can configure top and bottom menus with custom links, pages or categories. The top menu is the main menu with the main content like access to your business areas. The bottom menu let you publish other links like RSS feed, your Twitter account...

i18n: Theme in english and spanish

Boopress support internationalisation (i18n) with support to english and spanish languages.

Widget

Widget placed in two columns at the bottom, near of the footer.
